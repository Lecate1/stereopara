
# Система компьютерного зрения для стереопары на Orange Pi

## Описание проекта
Цель проекта — разработка программного обеспечения на C++ для одноплатного компьютера Orange Pi, которое позволяет захватывать и синхронно обрабатывать видеопоток с двух USB-камер (стереопары) и построения карты глубины (disparity map). Проект является исследовательским и направлен на изучение взаимодействия с вычислительными возможностями SoC под управлением ОС Linux.

**Ключевые особенности:**
*   Захват видео с двух камер с использованием OpenCV. 
*   Использование библиотеки OpenCV для базовой обработки изображений.
*   Калибровка стереопары.
*   Использование алгоритма для построения карты глубины. 
*   Сборка проекта с помощью системы Makefile.

## Аппаратное обеспечение
*   Одноплатный компьютер: **Orange Pi** (любая модель с USB-портами, например Orange Pi Zero 2 или Orange Pi 5)
*   Камеры: **2xUSB-камеры** (желательно одинаковые, с поддержкой UVC)
*   ОС: **Ubuntu** или официальная ОС от Orange Pi

## Программное обеспечение (Software)
*   **Язык:** C++
*   **Ключевые библиотеки:**
    *   **OpenCV** 
    *   Стандартная библиотека C++
